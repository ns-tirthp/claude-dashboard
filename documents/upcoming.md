  - Date/time range picker
  - Project comparison side-by-side
  - Search sessions by content or tool type
  - Custom metric dashboards (drag-drop widgets)

  - Multi-user usage aggregation
  - Shared project insights export
  - Cost allocation by team member or project tag

  - AI-powered chat interface to query telemetry via NL ("show me expensive prompts last week")

  Integrations:
  - Slack/email alerts for cost thresholds

  - CSV/JSON export for external analysis
  - Webhook support for custom workflows

  - Heatmaps (activity by hour/day)
